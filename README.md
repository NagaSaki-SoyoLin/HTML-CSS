# HTML/CSS 新闻页面学习项目

从零开始学习 HTML 与 CSS，以「央视新闻」页面为案例，逐步还原一个完整的新闻详情页。

## 📖 项目简介

本项目记录了 HTML/CSS 的入门学习过程，从最基础的标签开始，逐步深入到 CSS 选择器与样式排版，最终完成一个包含标题、正文、图片和视频的新闻页面还原。

学习参考页面为央视网《【新思想引领新征程】推进长江十年禁渔 谱写长江大保护新篇章》（2024年05月15日）。

## 🗂️ 项目结构

```
HTML-CSS/
├── 01. HTML快速入门.html                  # HTML 基础标签入门
├── 02. HTML快速入门.html                  # HTML 基础标签练习
├── 03. 央视新闻-标题-排版.html             # 新闻标题 + 超链接排版
├── 04. 央视新闻-标题-样式.html             # 标题样式（标签选择器）
├── 05. 央视新闻-标题-样式(选择器).html      # 标题样式（class/id 选择器）
├── 06. 央视新闻-正文-排版.html             # 完整新闻页（视频 + 图片 + 段落）
├── css/
│   └── news.css                          # 新闻页面样式表
├── img/                                  # 页面图片资源
├── 央视新闻原页面.txt                      # 原新闻页面参考文本
└── README.md
```

> 注：`video/` 和 `audio/` 目录因文件体积较大，未纳入版本管理。如需运行 `06` 页面，请自行添加 `video/news.mp4`。

## 🚀 快速开始

直接用浏览器打开任意 `.html` 文件即可查看效果：

```bash
# 例如查看完整新闻页面
start "06. 央视新闻-正文-排版.html"
```

## 📝 学习内容

| 阶段 | 知识点 |
|------|--------|
| 01-02 | HTML 基本结构、标题标签 `<h1>`、图片标签 `<img>` |
| 03 | 超链接 `<a>`（href、target 属性）、标题排版 |
| 04 | CSS 标签选择器、文字颜色、ID 选择器 |
| 05 | class 与 id 选择器、超链接去下划线 |
| 06 | 视频标签 `<video>`、段落标签 `<p>`、综合排版 |

## 🛠️ 技术栈

- HTML5
- CSS3
- 纯静态页面，无需构建工具
